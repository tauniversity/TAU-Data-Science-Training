Course Overview

Welcome to the AI General Assembly Training Programme at Thomas Adewumi University, Nigeria. The curriculum is meticulously designed to equip you with essential 21st-century skills, ensuring you are well-prepared for the evolving job market. Below is an introduction to each of the modules offered in this programme:

Data Analytics using Excel

Unlock the Power of Spreadsheets

Excel remains a cornerstone tool in data analysis, renowned for its versatility and robust functionality. This course delves deep into Excel’s advanced features, enabling you to manage, analyze, and visualize data effectively.

What You’ll Gain:

  • Mastery of Excel formulas and functions for complex calculations

  • Proficiency in creating and managing pivot tables and pivot charts

  • Skills in data cleaning, transformation, and preprocessing

  • Techniques for automating data cleaning tasks using Power Query.

  • Ability to design dynamic dashboards for insightful data visualization

Data Analytics using R

Harness the Potential of Statistical Computing

R is a powerful language tailored for statistical analysis and data visualization. This course provides a comprehensive introduction to R, equipping you with the tools to perform sophisticated data analyses and create compelling visual representations.

What You’ll Gain:

  • Understanding of R programming fundamentals and syntax

  • Expertise in data manipulation using packages like dplyr and tidyr

  • Skills in statistical modeling and hypothesis testing

  • Ability to create advanced visualizations with ggplot2

  • Knowledge of reproducible research practices using Quarto

Machine Learning with Python

Build Intelligent Systems

Python is the go-to language for machine learning and artificial intelligence. This course introduces you to the foundational concepts of machine learning, guiding you through the development of predictive models and intelligent applications.

What You’ll Gain:

  • Solid grasp of Python programming for data science

  • Understanding of key machine learning algorithms and their applications

  • Skills in data preprocessing, feature engineering, and model evaluation

  • Experience with popular libraries such as scikit-learn, TensorFlow, and Keras

  • Ability to deploy machine learning models in real-world scenarios

Power BI

Transform Data into Actionable Insights

Power BI is a leading business analytics tool that empowers you to visualize data and share insights across your organization. This course teaches you how to leverage Power BI to create interactive reports and dashboards that drive informed decision-making.

What You’ll Gain:

  • Proficiency in connecting and importing data from various sources

  • Skills in data modeling and relationship management

  • Ability to design and customize interactive visualizations

  • Techniques for sharing and collaborating on Power BI reports

  • Knowledge of Power BI service and its integration capabilities

Mastering Educational AI Tools

Unlocking AI Power: Essential Tools for Productivity and Creativity

Artificial Intelligence is at the forefront of revolutionizing various sectors by enhancing productivity, optimizing financial processes, and advancing healthcare solutions. This course introduces you to essential AI platforms such as ChatGPT, NotebookLM, Gemini, and Gamma, demonstrating their advanced applications to boost productivity and foster creativity.

What You’ll Gain:

  • AI Platforms Mastery: Learn to use leading tools like ChatGPT, NotebookLM, Gemini, and Gamma.

  • Boosted Productivity: Integrate AI to automate tasks and streamline workflows.

  • Creative AI Applications: Utilize AI for content creation, design, and innovative problem-solving.

  • Industry-Specific Solutions: Understand AI’s impact on finance, healthcare, and business operations.

  • Practical Experience: Apply AI tools through hands-on projects and real-world case studies.

  • Ethical AI Practices: Implement AI responsibly with a focus on ethical considerations.

LinkedIn Optimization

Enhance Your Professional Presence

LinkedIn is a vital platform for career advancement and networking. This course guides you through optimizing your LinkedIn profile to attract potential employers, showcase your skills, and expand your professional network effectively.

What You’ll Gain:

  • Strategies for creating a compelling LinkedIn profile

  • Techniques for showcasing your skills, experiences, and achievements

  • Insights into building and engaging with a professional network

  • Tips for leveraging LinkedIn features for job searching and career growth

  • Best practices for content sharing and personal branding on LinkedIn

Digital Marketing and Business Venture

Launch and Grow Your Digital Enterprise

In today’s digital age, effective marketing is crucial for business success. This course covers the essentials of digital marketing, equipping you with the skills to create, implement, and manage successful online marketing strategies for your business ventures.

What You’ll Gain:

  • Understanding of digital marketing fundamentals and strategies

  • Skills in search engine optimization (SEO) and search engine marketing (SEM)

  • Proficiency in social media marketing and content creation

  • Knowledge of email marketing, analytics, and performance measurement

  • Insights into launching and managing digital business ventures

Software Installation

Note

The hyperlinks featured on this webpage navigate to the official download pages for the respective software applications. After installation, please initiate the software to confirm its proper functionality and ensure that it launches and operates as expected.

Excel and Google Sheets

Microsoft Excel and Google Sheets are powerful spreadsheet tools widely used for data analysis, visualization, and complex calculations. Excel, part of the Microsoft Office suite, is renowned for its advanced features and robust functionality. Google Sheets, accessible through a web browser, offers real-time collaboration and compatibility across devices.

Please ensure your computer has Office 365 or Office 2016 installed for Excel. For Google Sheets, access is available through your university email.

Power BI

PowerBI is a powerful business analytics tool from Microsoft that enables users to visualize data, generate reports, and glean insights for informed decision-making.

To install PowerBI, please follow this link and select the file named PBIDesktopSetup_x64.exe to begin the installation process.

R Programming

R Programming is a powerful language and environment for statistical computing and graphics. It is widely used among statisticians and data analysts for developing statistical software and data analysis. R offers a variety of statistical and graphical techniques, and is highly extensible.

  • Install R
  • Install RStudio

R programming can be installed on any operating system (OS):

For Windows Users :

To install R on a Windows operating system, please download the setup file from this link. Once the download is complete, open the .exe file to begin the installation process.

For Mac Users :

To download R for Mac, please click the following link. Once the download is complete, open the .pkg file to begin the installation process.

For Linux Users:

To install the complete R System on Linux, please follow the instructions provided on the CRAN website.

For Ubuntu users with Apt-get installed, open your terminal and enter the following command: sudo apt-get install r-base.

To install RStudio, please follow this link. Select the appropriate installer for your operating system, download the file, and run the installer to complete the setup of RStudio.

Python

Python is a versatile, high-level programming language known for its simplicity and readability, making it ideal for beginners and experts alike. It supports multiple programming paradigms and is widely used in web development, data analysis, artificial intelligence, scientific computing, and more. Python’s extensive libraries and community support enhance its functionality for a broad range of applications.

Anaconda Distribution

Anaconda simplifies package management and deployment for Python and R, offering a suite of over 1,500 data science packages. It’s an indispensable tool for maintaining consistent environments across various data science projects.

To get started, please install the Anaconda distribution from the following link.

VS Code

Visual Studio Code (VS Code) is a lightweight but powerful source code editor that runs on your desktop. It comes with built-in support for JavaScript, TypeScript, and Node.js and has a rich ecosystem of extensions for other programming languages such as Python, PHP, and C++, among others. It’s designed to be fast and efficient, yet highly customizable to fit the needs of any developer.

To set up your environment, please install Visual Studio Code by following the link here and select the version appropriate for your operating system.

Git Bash

Git Bash provides a Unix-like command line environment for Windows users, enabling them to use Git for version control seamlessly. It includes common Bash utilities and Git functionality, making it a crucial tool for developers managing source code.

To install this version control system, please follow this link.
